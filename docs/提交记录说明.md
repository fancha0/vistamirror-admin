## 不建议直接改旧提交的原因

这些英文提交大多已经推送到远端。如果直接把历史提交标题改成中文，需要重写 Git 历史并强制推送。这样会改变提交 hash，可能影响 GitHub 记录、Docker 发布链路，或者让其他设备上的仓库需要重新同步。

因此，本项目默认用这份中文对照文档来解释历史提交，而不是改写历史。
